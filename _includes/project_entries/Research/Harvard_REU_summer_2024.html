research

The goal of this project is to advance bioprinting as a route to engineered tissues by developing a hydrogel bioink that supports both structural integrity and cell attachment. Working in the Lewis Lab through the Harvard REU program with support from MSREC and the NSF, this work builds on efforts to create synthetic tissues using multi-material 3D printing, including ValveJet approaches that combine extrusion and inkjet deposition. We focus on formulating a chitosan–gelatin–phytate double-network system with shear-thinning properties suitable for printing. The primary ink contains the chitosan–gelatin network, while a complementary Pluronic–phytate sacrificial ink both templates the printed geometry and crosslinks the C–G network to create stable, cell-compatible structures.
